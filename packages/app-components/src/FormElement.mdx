# FormElement

A wrapper component that provides consistent label styling and layout for form inputs. Use `FormElement` to wrap `TextInput`, `TextArea`, or `Select` components when you need a vertical label layout.

## Accessibility

`FormElement` includes built-in accessibility features:

- Uses semantic `<label>` element with `htmlFor` attribute
- Adds `aria-required="true"` when `required` prop is true
- Adds `aria-invalid="true"` when `error` prop is provided
- Links hint and error messages with `aria-describedby`
- Error messages have `role="alert"` for screen reader announcements

## Styling

The component uses CSS variables for theming:

- `--base-fg` - Label text color
- `--base-fg-dim` - Hint text color
- `--danger-fg` - Required indicator and error color

## Migration from Custom Labels

### Before
```tsx
<div className={styles.field}>
  <label>Name <span className={styles.required}>*</span></label>
  <TextInput value={name} onChange={setName} />
</div>
```

### After
```tsx
<FormElement label="Name" required>
  <TextInput value={name} onChange={setName} />
</FormElement>
```
