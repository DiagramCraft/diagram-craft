# MultiSelect

A searchable multi-select component that allows users to select multiple items from a predefined list. Features tag-like
selection with search and keyboard navigation capabilities.

## Features

- **Search and filter**: Type to search through available options
- **Tag-style selection**: Selected items appear as removable tags
- **Keyboard navigation**: Arrow keys, Enter, Escape, and Backspace support
- **Prevent duplicates**: Cannot select the same item twice
- **Smart filtering**: Only shows unselected items in suggestions
- **Responsive design**: Adapts to container size with appropriate tag width limits
- **Portal-based dropdown**: Properly positioned suggestions that work in complex layouts
