# Key Features

Diagram Craft starts as a fast canvas editor, but its feature set is aimed at diagrams that evolve beyond a quick sketch. You can move from basic boxes and arrows to richer notation systems, structured multi-tab documents, automated layouts, collaboration, and data-driven diagrams without switching to another tool.

## Interactive Diagram Editing

The core experience is direct manipulation on a large canvas:

- place shapes from built-in libraries and stencils
- draw connectors between elements
- move, resize, align, and distribute selections
- zoom and pan freely while working on large diagrams

This makes Diagram Craft suitable for both rough exploration and more precise technical diagramming.

## Rich Shape and Stencil Support

You can work with basic shapes or choose from more specialized libraries depending on the kind of diagram you are building.

Common use cases include:

- general block diagrams and flow-style visuals
- software and infrastructure diagrams
- architecture notation such as C4-style views
- domain-specific diagrams using custom or built-in stencils

This lets teams standardize their visual language instead of rebuilding the same symbols repeatedly.

## Styling, Effects, and Visual Refinement

Diagram Craft includes a deeper styling system than a minimal whiteboard tool. You can refine diagrams with:

- fills, strokes, and typography changes
- connector styling and arrowhead options
- shadows, gradients, and other visual effects
- geometric operations for building custom shapes

The result is a tool that can support both working diagrams and more presentation-ready output.

## Structure for Larger Diagrams

Once a diagram becomes more complex, the editor provides structure instead of forcing everything onto a single flat canvas.

Key organization features include:

- layers for separating and controlling parts of a diagram
- grouping for easier manipulation
- multiple tabs within the same document
- document structure views for navigating larger files

These features are especially useful when diagrams are maintained over time rather than created once and discarded.

## Layout and Editing Assistance

Diagram Craft includes tools that help clean up diagrams instead of requiring every position to be adjusted manually.

Examples include:

- alignment and distribution commands
- snapping to guides, grids, and nearby elements
- automatic layouts for structured graphs and hierarchies
- geometry operations for combining and refining shapes

This is where the editor starts to feel more like a diagramming system than a simple drawing surface.

## Collaboration and Review

Diagram Craft is built with collaborative work in mind. The project supports workflows such as:

- multi-user editing
- comments and review discussion
- awareness of other users in the same diagram
- version-oriented collaboration features

If your diagrams are shared artifacts rather than personal scratch work, these features matter as much as the drawing tools themselves.

## Data-Driven and Model-Driven Workflows

One of Diagram Craft's more advanced directions is linking diagrams to data and structured sources instead of keeping them entirely manual.

That includes capabilities such as:

- working with external data sources
- binding diagram content to data
- updating diagrams dynamically when data changes
- querying or transforming structured information for visual output

This opens the door to diagrams that stay synchronized with the systems they describe.

## AI and Text-Based Creation

Diagram Craft also explores workflows that reduce manual diagram construction:

- generating diagrams from text
- converting diagrams back into text representations
- using AI-assisted tooling where supported

These features are useful when you want to bootstrap a first draft quickly, document an existing diagram in another format, or iterate on structure before spending time on presentation details.

## Import and Export

A diagramming tool only becomes useful in practice if it fits into the rest of your workflow. Diagram Craft supports that with import and export options such as:

- importing existing Draw.io work
- exporting diagrams as images or SVG
- saving work in its native format for continued editing

This makes it easier to adopt Diagram Craft without starting from zero every time.

## Open Source and Extensible

Diagram Craft is open source and organized as a set of packages rather than a single closed application. That makes it relevant both for end users and for teams that want to:

- self-host the editor
- extend it with custom behavior
- build on top of the underlying packages
- tailor stencil libraries or workflows to their own domain

If you only need the editor, you can ignore the implementation details. If you need deeper control, the project is structured to support that too.
