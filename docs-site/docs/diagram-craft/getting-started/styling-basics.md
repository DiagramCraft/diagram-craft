# Styling Basics

Once the structure of the diagram is in place, styling helps communicate meaning and make the diagram easier to scan.

## Start With The Basics

For a first pass, focus on four controls:

- **fill color** to separate groups or emphasis levels
- **stroke color** to strengthen or soften outlines
- **stroke width** to change visual weight
- **text styling** to improve label readability

These changes are usually enough to move from a rough draft to a presentable diagram.

## A Simple Styling Workflow

1. Select one shape or a small group of related shapes.
2. Apply a fill color that supports meaning rather than decoration.
3. Adjust the stroke if the outline is too faint or too heavy.
4. Make sure text remains readable against the fill color.
5. Repeat the same style choices for similar elements.

Consistency matters more than variety. A small, repeated style system reads better than many one-off visual tweaks.

## When To Add Effects

Effects such as shadows can help separate elements from the background, but they are best used lightly. For early diagrams:

- use one subtle shadow style if it improves depth
- avoid stacking multiple effects on the same element
- keep the diagram readable in export formats
