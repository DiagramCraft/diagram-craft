# Introduction

Diagram Craft is a browser-based diagram editor for building technical and visual diagrams without switching tools as your work becomes more detailed. You can start with a few basic shapes, then move into richer stencils, styling, layering, import/export, and more advanced editing when you need it.

This getting-started section is aimed at first-time users of the editor. It focuses on the shortest path to a usable diagram rather than covering every feature.

## What You Will Learn

By the end of this section you should be able to:

- open Diagram Craft and recognize the main editing areas
- place shapes on the canvas and connect them
- apply a few basic style changes
- save your work and export a shareable image
- find the next set of docs based on what you want to do next

## Core Ideas

Before you begin, it helps to know a few terms:

- The **canvas** is the large editing surface where you place and arrange diagram elements.
- **Shapes** are the nodes in your diagram, such as rectangles, circles, containers, or stencil-specific elements.
- **Connectors** are the lines or arrows between shapes.
- **Layers** help you organize or separate parts of a diagram without creating a new file.
- A **document** can contain more than one diagram tab.

You do not need to memorize the terminology up front. The first walkthrough introduces these ideas in context.
