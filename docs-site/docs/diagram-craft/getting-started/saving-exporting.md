# Saving and Exporting

After you have a workable diagram, the next step is preserving it in a form you can revisit or share.

## Save The Diagram Document

The native document format keeps the editable structure of the diagram, including:

- shapes and connectors
- diagram tabs
- layers
- styling information

Use the native document when you want to keep editing later.

## Export For Sharing

When you need an output for presentations, chat, tickets, or documentation, export the diagram as an image:

- **PNG** is useful for screenshots, slide decks, and general sharing.
- **SVG** is useful when you want a scalable vector result.

Export after checking that the diagram is framed well on the canvas and that labels are readable at the intended size.

## Reopen Existing Work

If you are working with a saved document, reopen that document rather than rebuilding the diagram from an exported image. The export is a presentation artifact, not the editable source of truth.
