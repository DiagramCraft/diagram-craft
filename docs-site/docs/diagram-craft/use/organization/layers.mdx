# Layers

Layers control element stacking, visibility, and editing access inside the active diagram. Manage them from the **Layer** tab in the left-side **Structure** window.

## Add And Reorder Layers

Use the layer actions in the Structure window to create a regular, rule, or reference layer. Drag layers in the list to change their order; that order determines how their content is stacked on the canvas.

Selecting a layer makes it active. Newly created elements are added to the active regular layer.

## Hide And Lock Layers

Use the visibility action to hide or show all content on a layer. Use the lock action to prevent its elements from being changed through normal canvas editing. Both states are controlled independently from the layer row.

## Inspect Layer Contents

Expand a layer in the Structure window to inspect its nodes, edges, and nested groups. Select an item in the tree when direct selection on a crowded canvas is difficult.

## Use Rule And Reference Layers

The layer stack can hold more than plain drawing layers:

- **Reference layers** help reuse context from another diagram without copying everything into the current canvas.
- **Rule layers** are useful when visibility or styling depends on diagram data or query results.

Rule and reference layers have different configuration from regular drawing layers. Select the layer in the Structure window to inspect the properties available for its type.
