# Groups

Groups combine selected elements into one node that can be selected and moved as a unit.

## Create A Group

Select at least two compatible elements, open the selection context menu, and choose **Group**. Diagram Craft creates a group node and moves the selected elements into it while preserving their canvas positions.

## Select And Move A Group

Click grouped content to select the group itself. Moving the group updates the grouped contents together. The group and its children are also visible in the active layer's element tree.

## Edit Group Contents

Select within the group when you need to change an individual child. Alignment and distribution commands can operate on the children of a single selected group when it contains enough elements.

## Nest Or Ungroup

Groups can contain other groups. To remove a group container, select it and choose **Ungroup** from the context menu. The children return to the parent layer while keeping their current positions.
