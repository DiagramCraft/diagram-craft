# Document Structure

The Structure Panel is the control surface for keeping a complex document tidy. Use it when the canvas alone is no longer enough for navigation, reordering, or understanding hierarchy.

## Two Different Views

The Structure Panel has two especially useful tabs for diagram scale:

- **Layer** shows the active diagram's layer stack and element tree.
- **Document** shows the hierarchy of diagrams and subpages across the document.

Use the Layer tab when you are managing the current canvas. Use the Document tab when you are managing the document as a whole.

## Navigate From The Structure Window

Expand tree items to inspect nested diagrams, layers, groups, and elements. Selecting an entry navigates to or selects the corresponding document object, which is useful when it is difficult to locate directly on the canvas.

## Use The Document Tab

The Document tab is especially useful when:

- you need to move between overview and detail pages quickly
- a review reveals that one diagram should become a subpage of another
- tab names or ordering no longer match the story the document tells

It gives you a tree view of the same document hierarchy that the tab strip only shows partially.

## Use The Layer Tab

The Layer tab helps when:

- a crowded diagram makes direct selection difficult
- you need to lock or hide one concern while editing another
- you want to inspect nested elements or grouped content in context

That is often faster than clicking around the canvas at high zoom levels.
