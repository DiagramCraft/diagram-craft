# Tabs and Documents

Each Diagram Craft document can contain multiple diagrams. The tab strip provides quick access to them, while the **Document** tab in the **Structure** window exposes their complete hierarchy.

## Add And Select Diagrams

Use the add action beside the tab strip to create a diagram. Select its tab to make it active; the canvas, active layer, and relevant tool windows update to the selected diagram.

## Create Nested Diagrams

Diagram Craft supports nested diagrams. That is useful when one top-level diagram owns a small family of supporting views.

Open a diagram's context menu and choose **Add subpage** to create a nested diagram. Nested diagrams remain independent canvases; nesting changes document navigation rather than grouping their elements.

## Manage The Document Tree

Use the **Document** tab in the Structure window to navigate, rename, delete, and reorder diagrams. Drag-and-drop reordering works between diagrams with the same parent. The tree exposes nested diagrams even when the tab strip cannot show the full hierarchy at once.
