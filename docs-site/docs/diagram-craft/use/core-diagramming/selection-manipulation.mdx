# Selection and Manipulation

Selection is the entry point to most editing work. Once the right shapes or connectors are selected, you can move them, resize them, duplicate them, or clean up a section of the diagram in a single pass.

## Select Elements

Diagram Craft supports both direct selection and area selection.

- click an element to select it directly
- click-drag on empty background to create a marquee selection
- hold `Shift` while clicking to add or remove elements from the current selection
- hold `Shift` while marquee-dragging to add that marquee result to the current selection
- use `Command+A` to select everything

This lets you work with a single element, a local cluster, or the whole diagram depending on the edit you are making.

## Move A Selection

Once an element or group is selected, drag it to reposition it.

- drag normally for a snapped move
- hold `Shift` while dragging to constrain movement to a horizontal or vertical axis based on the initial drag direction
- hold `Option` while dragging to disable snapping
- hold `Command` while dragging to duplicate the current selection and move the duplicate

You can also move the selection from the keyboard:

- `Arrow keys` move the selection by one step
- `Shift+Arrow keys` move the selection by the grid step

## Resize A Selection

Resize handles appear on the selection when the current element type supports resizing.

- drag a resize handle to change width or height
- hold `Shift` while resizing to constrain the aspect ratio
- hold `Option` while resizing to disable snapping

Keyboard resizing is also available:

- `Command+Arrow keys` resizes by one step
- `Command+Shift+Arrow keys` resizes by the grid step

## Rotate A Selection

When the selected element supports rotation, use the rotation handle to change its angle.

- drag the rotation handle for free rotation with angle snapping
- hold `Option` while rotating to disable snapping

By default, rotation snaps to 5-degree increments.

## Duplicate And Delete

Selection also controls quick structural edits:

- `Command`-drag duplicates the current selection as part of a move
- `Command+D` duplicates the current selection without dragging
- `Backspace` deletes the selection

These commands work for single elements and multi-selection alike.
