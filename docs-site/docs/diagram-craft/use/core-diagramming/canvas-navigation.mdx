# Canvas Navigation

Canvas navigation is what keeps the editing loop fluid. You should be able to zoom out to regain context, pan to the next area you want to refine, and then zoom back in without losing your place in the diagram.

## Start With Zoom And Pan

Most navigation in Diagram Craft comes down to two actions:

- zoom to change how much of the diagram you can see
- pan to move the current viewport without moving any diagram elements

Use zoom when the current level is wrong for the task. Zoom out to check the overall flow, then zoom in again when you need to place a connector precisely or edit a label.

Use pan when the scale is already right and you just need to move to another part of the canvas.

## A Reliable Navigation Workflow

When a diagram starts to spread out, this pattern keeps you oriented:

1. Zoom out until the current area and the next target area are both visible.
2. Pan toward the part of the diagram you want to edit next.
3. Zoom back in only when you need more precision.
4. Make the edit, then zoom out again to verify that it still fits the larger layout.

This is usually faster than staying zoomed in and dragging the viewport around one screen at a time.

## Regain Context When You Feel Lost

If the diagram no longer fits comfortably in view:

- zoom out until the main structure is visible again
- use the canvas frame and rulers to understand where the current work sits
- return to a comfortable zoom level before continuing detailed edits

Treat zooming out as a quick orientation step, not as a sign that something went wrong. It is a normal part of working on larger diagrams.

## Use Navigation To Check Readability

Navigation is also a review tool. Before exporting or sharing a diagram:

- zoom out to see whether the main flow is still obvious
- pan across the diagram looking for cramped labels or awkward crossings
- zoom back in to correct the places that feel dense

If the diagram reads clearly at more than one zoom level, it is usually in good shape.
