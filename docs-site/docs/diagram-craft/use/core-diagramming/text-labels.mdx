# Text and Labels

Diagram Craft supports two common text workflows: standalone text placed directly on the canvas, and labels attached to other elements such as edges.

## Use Standalone Text

Standalone text is the free text you place directly on the canvas.

- use the **Text tool** from the main toolbar when you want to place text directly
- use the default shortcut `Command+4` to switch to the text tool from the keyboard
- click on the canvas to place a standalone text element
- you can also drag a text-capable shape from the shape browser when that fits your workflow better

This is the right path for titles, notes, section headings, and other text that does not belong to a single connector or node.

## Add And Edit Edge Labels

Edge labels are attached to a connector rather than floating independently on the canvas.

- double-click an edge when you want to edit its text in place
- use the edge context menu when you want to add text explicitly through the **Add text** action
- use **Edit Text** from the context menu when the selected element already exposes editable text fields

This is useful for relationship names, flow conditions, protocol labels, or short annotations that belong to a connector itself.

## Edit Text In Place

Most text editing in Diagram Craft happens directly on the canvas.

- double-click the target text when you want to edit it without leaving the canvas
- use the context menu when the element exposes multiple text fields or a text-specific action
- reposition or resize the surrounding element after editing if the text no longer fits cleanly

This keeps text editing close to the diagram structure you are already working on.

## Refine Text After Placement

After the wording is correct, refine the visual fit:

- resize the containing shape if a label wraps awkwardly
- move standalone text when it overlaps the main flow
- use the styling panels after placement when you need stronger visual distinction
