# Drawio Import

Draw.io import is the fastest way to move an existing diagram into Diagram Craft when redrawing would cost more than cleanup. Use it for migration, reuse of older assets, or as a starting point for a diagram you want to keep evolving in Diagram Craft.

## What Usually Transfers Well

Import is designed to preserve the working structure of the source diagram:

- the overall node and edge structure
- many common shapes
- a large part of the existing layout
- labels and a useful amount of styling

That makes it a good migration starting point rather than a blank reimplementation.

## Where To Expect Cleanup

Treat the import as a conversion, not a pixel-perfect clone. After opening the file, review:

- labels that may wrap differently
- connectors that need rerouting or spacing cleanup
- containers, swimlanes, or layout-driven regions
- imported Draw.io stencil shapes that should be checked visually

## A Good Migration Loop

1. Import the `.drawio` file.
2. Review the overall structure before changing styling.
3. Fix any layout or labeling issues that affect readability.
4. Convert the result into your normal Diagram Craft editing workflow.

## When It Is A Good Fit

- you want to keep the logical structure of an existing Draw.io diagram
- the source diagram is large enough that redrawing it is wasteful
- the imported file is a starting point for further editing, not the final deliverable

## When It Is Not

If the original diagram depends heavily on exact visual fidelity or very tool-specific behavior, expect more manual cleanup. In those cases, import still saves time, but it should be treated as assisted migration rather than a finished result.
