# Layout Configuration

Each layout command opens a dialog with the parameters supported by that algorithm. Parameter changes update the canvas preview before you apply the result.

## Tree, Layered, Orthogonal, And Series-Parallel

These layouts expose the same core controls:

- **Gap** controls how much space Diagram Craft leaves between nodes or levels.
- **Direction** controls whether the result reads down, up, left, or right.

Changing **Gap** adjusts separation between nodes or levels. **Direction** supports down, up, left, and right.

## Force-Directed

Force-Directed exposes three different controls:

- **Spring** changes how strongly connected nodes pull toward each other
- **Repulsion** changes how strongly nodes push away from each other
- **Length** changes the target edge length

Adjusting these values updates the preview so you can compare the result before applying it.

## Apply The Configuration

Choose **Ok** to commit the previewed positions as one undoable edit. Choose **Cancel** to discard the preview.
