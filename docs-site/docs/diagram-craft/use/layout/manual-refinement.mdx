# Manual Refinement

After applying an automatic layout, use Diagram Craft's selection, alignment, snapping, and connector controls to adjust individual areas without rerunning the whole layout.

## Refine A Local Selection

1. Check the overall reading order first.
2. Move or regroup any local cluster that is telling the wrong story.
3. Use alignment and distribution on repeated shapes.
4. Use snapping and guides to normalize spacing.
5. Re-run automatic layout only on the area that still needs structural help.

Automatic layout commands can also be run on a smaller selected region when it needs a different arrangement.

## Adjust Connectors

Select an edge to change its type, move waypoints, or adjust bezier handles after the connected nodes have moved. Edge labels remain attached to the connector and can be repositioned as part of the same cleanup pass.
