# Manual Refinement

Automatic layout gets you to a strong draft. Manual refinement is what makes the diagram publishable.

## What To Refine After Auto Layout

Focus on the places where human judgment matters:

- the main narrative path
- crowded edge crossings
- label spacing
- repeated blocks that should look intentionally aligned
- local clusters that deserve tighter composition than the global layout can infer

## A Practical Cleanup Sequence

1. Check the overall reading order first.
2. Move or regroup any local cluster that is telling the wrong story.
3. Use alignment and distribution on repeated shapes.
4. Use snapping and guides to normalize spacing.
5. Re-run automatic layout only on the area that still needs structural help.

This is usually faster than repeatedly relaying out the entire diagram.

## Combine Layout With Organization Tools

Large-diagram maintenance works best when these tools are used together:

- **tabs and subpages** for scope
- **layers** for visibility and lock control
- **groups** for local manipulation
- **automatic layout** for structural recovery
- **manual cleanup** for readability

That is the real workflow. The tools help each other.

## Know When To Stop

If a section still fights you after multiple layout passes, it is often a decomposition problem rather than a positioning problem. Split the content into another tab, hide optional detail on another layer, or simplify the local cluster before continuing.
