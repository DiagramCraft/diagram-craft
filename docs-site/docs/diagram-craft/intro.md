# Introduction to Diagram Craft

Welcome to Diagram Craft documentation!

Diagram Craft is an interactive diagramming tool that allows you to create professional diagrams with ease.

## What is Diagram Craft?

Diagram Craft is a powerful, browser-based diagramming application that provides:

- **Intuitive Drawing Tools**: Create shapes, connectors, and complex diagrams
- **Real-time Collaboration**: Work together with your team in real-time
- **Advanced Features**: Layer management, boolean operations, custom shapes
- **Import/Export**: Support for various formats including DrawIO
