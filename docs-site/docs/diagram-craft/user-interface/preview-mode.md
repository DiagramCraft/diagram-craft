# Preview Mode

Preview mode strips the editor down to the diagram and a small control set so you can review or present without the normal editing chrome.

## Enter Preview Quickly

You can enter preview from the auxiliary toolbar or by using the default shortcut `Command+Shift+P`.

## What Changes In Preview

Preview mode replaces the main editor chrome with:

- zoom controls
- zoom-to-fit
- fullscreen entry and exit
- close

The result is a cleaner viewing surface that is easier to use during walkthroughs, reviews, or screenshot capture.

## Fullscreen

Preview mode can request fullscreen directly from its own controls. That is the best option when you want a presentation-like view without browser chrome or surrounding UI.

## Exit Paths

- use the close button in preview
- leave fullscreen from the preview controls
- press `Escape`
