# Your First Workspace

## What a workspace does

A workspace groups schemas, entities, projects, teams, and settings.
It is the main boundary for data in Arch Register.

## Create a workspace

<ThemedImage
  alt="Workspace selector in Arch Register"
  sources={{
    light: require('/img/arch-register/workspace/selector-open-light.png').default,
    dark: require('/img/arch-register/workspace/selector-open-dark.png').default,
  }}
/>

1. Open the workspace selector in the top bar.
2. Choose `New workspace...`.
3. Enter a name and optional description.
4. Leave the mode on blank for a first workspace unless you need a template or a copy.
5. Select `Create workspace`.

<ThemedImage
  alt="Create workspace dialog in Arch Register"
  sources={{
    light: require('/img/arch-register/workspace/create-dialog-light.png').default,
    dark: require('/img/arch-register/workspace/create-dialog-dark.png').default,
  }}
/>

## What happens next

After creation, Arch Register opens the new workspace and updates the workspace list.
