# Create Your First Entity

## Before you start

You need at least one schema in the workspace.
The default bootstrap data includes schema types you can use for a first entity.

## Create an entity

1. Open **Entities**.
2. Select `New entity`.
3. Pick a schema type.
4. Enter the entity name.
5. Fill in any required fields.
6. Select `Create entity`.

The same dialog is available from the home screen when entity creation is enabled.

![Create entity dialog](/img/arch-register/entities/create-dialog.png)

## After creation

The new entity opens in its detail view.
From there you can edit data, review history, and navigate to related content.
