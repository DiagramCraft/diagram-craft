# Create Your First Project And Diagram

This guide introduces the content side of Arch Register: create a project, add a diagram, and open the editor.

## Before You Start

You should already have:

- access to a workspace
- permission to create or edit projects

## Create A Project

1. Open **Projects**.
2. Select **New project**.
3. Enter the project name.
4. Set owner, status, and any optional metadata.
5. Save the project.

The project opens in its detail view after creation.

## Add A Diagram

1. In the project, open **New**.
2. Select **New diagram**.
3. Pick **Blank canvas** or a template.
4. Enter the diagram name.
5. Create the diagram.

The diagram becomes part of the project's file tree and can be reopened from the project sidebar.

## What To Notice

The first diagram flow introduces several product concepts at once:

- projects are content containers
- folders are optional and can be added later
- templates can be local to a project or exposed more broadly
- diagram editing happens in the Diagram Craft editor, but the file is managed by Arch Register
