# Core Concepts

## Workspace

The workspace is the top-level container. It separates one set of architecture data from another.

## Schema

A schema defines an entity type and its fields.
The entity browser uses schemas as filters and the create entity dialog uses them as the starting point for new records.

## Entity

An entity is a record created from a schema.
Entities appear in the entity browser, in entity detail screens, and in search results.

## Project

A project groups diagrams, folders, and related files.
Projects are separate from entities, but can reference them.
Project diagrams can also be marked as templates and opened collaboratively.

## Team

Teams are used for ownership and permissions.
They are part of workspace configuration.
