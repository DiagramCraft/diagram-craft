# Entities

Entities are records created from schemas.

An entity carries the fields defined by its schema plus workspace metadata such as name, owner, lifecycle, and tags.

## Common actions

- Browse entities in the entity browser.
- Filter by schema, owner, lifecycle, and search query.
- Open an entity to review or edit it.
- Create a new entity from the browser or from the home screen.
