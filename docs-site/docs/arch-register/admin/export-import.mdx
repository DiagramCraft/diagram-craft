# Export And Import

Export and import are administrative operations because they move or reshape workspace data at a broader scope than
normal editing.

## Workspace-Level Operations

The workspace settings area includes export and import for moving data between workspaces or creating a packaged
snapshot.

## Entity-Level Operations

Entity CSV import and export are documented with the entity workflows because they are narrower operational tools tied
to the catalog browser.

## Keep The Distinction Clear

Use:

- workspace export/import for workspace-wide movement
- entity import/export for schema-oriented catalog operations
