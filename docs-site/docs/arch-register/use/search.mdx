# Search

Search is the cross-section discovery tool for the current workspace. Use it when you know what you need but not
where it lives.

## What Search Covers

Workspace search spans:

- entities
- projects
- diagrams
- wiki pages and other content
- schemas

This is broader than the local filters inside entity browsers or project folders.

## When To Use Search

Use search when:

- you do not know whether the item is an entity or project artifact
- you want to find a schema, record, or file from one entry point
- you need to jump across product areas quickly

Do not use workspace search when you are already inside a narrow slice and only need a local filter. In that case,
use the filter controls inside the current screen.

## Result Shape

Results are grouped by category and include enough context to show where the match belongs. That context is important
for project content, where the same file name may appear in more than one project or folder.
