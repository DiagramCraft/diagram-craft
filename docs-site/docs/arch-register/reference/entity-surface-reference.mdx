# Entity Surface Reference

This page is the compact map of the implemented entity surfaces.

## Entity Browser

Main capabilities:

- filter and facet browsing
- saved views
- multiple visual modes
- CSV export
- CSV import entry point

## Entity Detail Tabs

The entity detail experience includes:

- overview
- topology
- graph
- relationships
- change history
- timeline
- content

## Advanced Views

The entity browser supports:

- table
- cards
- tree
- radar
- timeline
- matrix
- bubble
- explore
