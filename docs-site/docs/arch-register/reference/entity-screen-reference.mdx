# Entity Screen Reference

This page is the compact map of the implemented entity screens.

## Entity Browser Screens

Main capabilities:

- filter and facet browsing
- saved views
- multiple screen modes
- CSV export
- CSV import entry point

## Entity Detail Screens

The entity detail experience includes:

- overview
- topology
- graph
- relationships
- assessments
- change history
- timeline
- content

## Browser Modes

The entity browser supports:

- table
- cards
- tree
- radar
- timeline
- matrix
- explore
