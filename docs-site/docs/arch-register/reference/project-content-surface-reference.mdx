# Project And Content Surface Reference

This page is the compact map of the implemented project and content surfaces.

## Project Surfaces

The project area includes:

- project list and archive views
- project detail
- folder tree navigation
- diagrams
- wiki pages
- file uploads
- templates
- pinned projects
- project-entity planning views

## Content Surfaces

Content editing exists in three parent contexts:

- workspace content
- project content
- entity content

Shared capabilities include:

- folders
- markdown editing
- attachments
- embedded references
- history and compare flows
