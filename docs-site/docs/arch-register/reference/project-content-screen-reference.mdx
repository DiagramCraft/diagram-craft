# Project And Content Screen Reference

This page is the compact map of the implemented project and content screens.

## Project Screens

The project area includes:

- project list and archive views
- project detail
- folder tree navigation
- project content
- associated entities
- assessments list
- assessment details with details and summary tabs
- project-planned changes

## Content Screens

Content editing exists in three parent contexts:

- workspace content
- project content
- entity content

Shared capabilities include:

- folders
- markdown editing
- diagrams
- attachments
- embedded references
- history and compare flows
